## Design Guidelines

### Colors
- Background: #f7f6f3 (beige, matching site)
- Primary: #37352f (dark gray)
- Accent: #2eaadc (blue)
- White: #ffffff

### Typography
- Font: Inter (same as website)
- Title: Bold, 48-60px
- Subtitle: Regular, 24-32px

### Layout
- Center-aligned content
- Adequate padding (80-100px on all sides)
- High contrast for readability
- Professional and clean aesthetic

## Tools for Creating Images

### Online Tools
- [Canva](https://www.canva.com/) - Easy drag-and-drop design

- Search for "Open Graph template" or "Social media preview template"
- Use 1200x630 dimensions for og-image
- Use 1280x720 dimensions for screenshot
